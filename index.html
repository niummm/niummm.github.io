UDF提权前提

  • 拥有能将Udf.dll写入相应目录的权限。
  • Mysql版本大于5.1，则Udf.dll文件需放至在Mysql安装目录的/lib/plugin文件夹下。
  • Mysql版本小于5.1，则Udf.dll文件在Windows server 2003中放至在C:/windows/system32目录。在Windows server 2000中放至在C:/winnt/system32目录下。
  • 掌握的Mysql数据库的账户，要拥有对Mysql数据库的Insert和Delete权限，以创建和抛弃函数。
  • 版本大于5.1的Udf.dll放至Mysql安装目录的/lib/plugin文件夹下才能创建自定义函数。目录是默认不存在，需要自己创建。在安装目录下创建/lib/plugin文件夹，将Udf.dll导入到这个文件夹下。

UDF.DLL文件提取

  • Udf.dll在Sqlmap中可以找到，在/sqlmap/data/udf/mysql/文件夹下，分为linux/windows又各分为32/64位。这里的位数是mysql的位数，并不是对方系统的位数

Sqlmap里的Udf.dll是经过编码的，需要先解码，解码工具在sqlmap/extra/cloak/cloak.py

解码完后的文件会生成在/sqlmap/udf/mysql/windows|linux/32|64文件下生成dll文件/
将dll文件复制到mysql的/lib/plugin目录下。

用十六进制编辑器打开Udf.dll，可以查看支持创建的函数。
